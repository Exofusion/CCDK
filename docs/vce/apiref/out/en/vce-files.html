vce.h
SummaryVCE header file for C language
Compatibilitylinux,win32,darwin
Descriptionvce.h includes a prototype declaration and an extern variable declaration for all functions of VCEgg API.
Notes for compiling under a C++ environment A compiler must be specified to define _the cplusplus macro. VCEgg's code is essentially written with UNIX's C so it has the "linkage" of C. Microsoft VisualStudio, for instance, has "C++ linkage" as default, so a prototype declaration by extern "C" is required to use functions with a C linkage.
